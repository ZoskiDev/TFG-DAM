Bienvenido a O5O, aqui unas introducciones basicas:

1. SETTEAR TOKEN

En el archivo bin/token.txt debes incluir la clave publica de tu bot para que el codigo de O5O se aplique

el archivo debe contener unicamente el contenido del token, en una sola linea.



2. INICIAR APP

Para que O5O funcione, debes ir a la carpeta /bin y ejecutar el script .bat (IMPORTANTE QUE TENGAS EL ARCHIVO TOKEN.TXT CON LA RESPECTIVA CLAVE)
